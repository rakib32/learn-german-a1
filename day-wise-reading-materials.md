## 🗓️ 2-Week Study Plan

### Day 1 — Greetings & Introductions

#### Key Words
| German | English |
|---|---|
| Hallo | Hello |
| Guten Morgen | Good morning |
| Guten Tag | Good afternoon |
| Guten Abend | Good evening |
| Auf Wiedersehen | Goodbye |
| Tschüss | Bye (informal) |
| Bitte | Please / You're welcome |
| Danke | Thank you |
| Ja | Yes |
| Nein | No |
| Entschuldigung | Excuse me / Sorry |

#### Key Phrases
| German | English |
|---|---|
| Wie heißen Sie? | What is your name? (formal) |
| Wie heißt du? | What is your name? (informal) |
| Ich heiße Rakibul | My name is Rakibul |
| Ich bin Rakibul | I am Rakibul |
| Woher kommen Sie? | Where are you from? (formal) |
| Ich komme aus Bangladesch | I come from Bangladesh |
| Wo wohnen Sie? | Where do you live? |
| Ich wohne in Berlin | I live in Berlin |
| Wie geht es Ihnen? | How are you? (formal) |
| Wie geht's? | How are you? (informal) |
| Gut, danke | Good, thank you |
| Nicht so gut | Not so good |

#### Grammar — Verb "sein" (to be)
| Pronoun | German | English |
|---|---|---|
| I | ich bin | I am |
| You | du bist | you are |
| He/She/It | er/sie/es ist | he/she/it is |
| We | wir sind | we are |
| You (plural) | ihr seid | you all are |
| They/You (formal) | sie/Sie sind | they/you are |
